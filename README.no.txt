Eltako har siden 1949 stått for bryting og byggautomasjon, med tyskproduserte installatørkvalitetsprodukter kjent for holdbarhet og lang levetid. Fra reléer og dimmere til sensorer og aktuatorer – Eltako-produkter er tillitsfulle i hele Europa for pålitelig, no-nonsense ytelse.

Denne appen bringer den utprøvde teknologien til Homey og kobler Eltako-enhetene dine sømløst sammen med resten av det smarte hjemmet. Samme kvalitet du forventer fra Eltako, nå integrert med Homey.

Støttede EnOcean-enheter (krever EOA64-modul)
----------------------------------------------
- F2T55E Trådløs 2-veis trykknapp
- F4T55E Trådløs 4-veis trykknapp
- FBHT55ESB-am Trådløs bevegelse/lysstyrke/temperatursensor (krever firmware v2.4.0)
- FTFSB Trådløs temperatur- og fuktsensor (krever firmware v2.4.0)
- FTKB Trådløs vindu-/dørkontakt (krever firmware v2.4.0)
- FTKE Trådløs vindu-/dørkontakt (krever firmware v2.4.0)
- FWS81 Trådløs vanndetektor (krever firmware v2.4.0)
